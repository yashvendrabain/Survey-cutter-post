### 7. Inspect a downloaded zip

Pick one of the SINGLE_SELECT downloads. The zip should contain exactly two files:
- bain_survey_template.pptx
- <question_id>_<label>.ppttc

Open the .ppttc in a text editor. It should be JSON with this exact shape:

[
  {
    "template": "bain_survey_template.pptx"
  },
  {
    "data": [
      {
        "name": "Chart_stacked_column",
        "table": [
          [{}, {"string": "Respondents"}],
          [{"string": "<option label>"}, {"number": 0.42}],
          ...
        ]
      }
    ]
  }
]

Critical checks:
- Top level is a LIST with exactly 2 entries (template and data as siblings, NOT nested)
- The corner cell [0][0] of the table is {} (empty object), NOT null
- "template" value is just the filename, not an absolute path
