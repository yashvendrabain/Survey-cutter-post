FIX 2 — Deprioritize imbalanced candidates in candidate_outcome_questions

In src/survey_type_detector.py, in _score_outcome_relevance(), the 
function has no awareness of actual data distribution. We cannot fix that 
there (it doesn't receive decoded_df). 

Instead, in detect_survey_type() in app.py wiring (the block in 
_section_survey_classification() that calls detect_survey_type()), after 
the result is computed, re-sort all_eligible_questions to penalise 
imbalanced questions:

After:
    survey_type_result = detect_survey_type(
        schema=st.session_state.schema,
        decoded_df=st.session_state.decoded_df
    )
    st.session_state.survey_type_result = survey_type_result
    st.session_state.outcome_variable_id = survey_type_result.outcome_question_id

Add:
    # Penalise imbalanced outcome candidates so they sort lower in dropdown
    from src.models import OutcomeVariableOption
    
    def _imbalance_penalty(opt, decoded_df):
        if opt.question_id not in decoded_df.columns:
            return 0.0
        col = decoded_df[opt.question_id].dropna()
        if len(col) == 0:
            return 0.0
        top_pct = col.value_counts(normalize=True).iloc[0]
        if top_pct >= 0.95:
            return 0.4   # severe imbalance — drop score significantly
        if top_pct >= 0.90:
            return 0.2   # moderate imbalance
        return 0.0
    
    rebalanced = sorted(
        survey_type_result.all_eligible_questions,
        key=lambda opt: opt.relevance_score - _imbalance_penalty(opt, st.session_state.decoded_df),
        reverse=True,
    )
    
    # Also update outcome_question_id if the top candidate after penalty differs
    best_id = rebalanced[0].question_id if rebalanced else None
    if best_id != survey_type_result.outcome_question_id:
        st.session_state.outcome_variable_id = best_id
    
    # Store rebalanced list back — we replace only all_eligible_questions
    # SurveyTypeResult is frozen so rebuild with updated field
    from dataclasses import replace
    survey_type_result = replace(
        survey_type_result,
        all_eligible_questions=rebalanced,
        outcome_question_id=best_id,
    )
    st.session_state.survey_type_result = survey_type_result
    st.session_state.outcome_variable_id = survey_type_result.outcome_question_id
